### Given an array of size N, the task is to check whether the array consists of consecutive numbers or not.

```java
import java.util.HashSet;

public boolean isConsecutive(int[] a) {
    int n = a.length;
    Set<Integer> uniqueElements = new HashSet<>();
    int min = Integer.MAX_VALUE;
    int max = Integer.MIN_VALUE;
    
    for (int ele : a) {
        min = Math.min(min, ele);
        max = Math.max(max, ele);
        
        if (uniqueElements.contains(ele)) return false;
        uniqueElements.add(ele);
    }
    
    return max - min + 1 == n;
}
```
