# Practice Stream API

## 1. Print all numbers given in a list on a new line.
```java
import java.util.Arrays;

class Main {
    public static void main(String[] args) {
        int[] a = {1,2,3,4,5,6,7,8,9};
        printNumbers(a);
    }
    
    private static void printNumbers(int[] a) {
        Arrays.stream(a).forEach(System.out::println);
    }
}
```

## 2. print all even elements of a given list
```java
import java.util.Arrays;

class Main {
    public static void main(String[] args) {
        int[] a = {1,2,3,4,5,6,7,8,9};
        printNumbers(a);
    }
    
    private static void printNumbers(int[] a) {
        Arrays.stream(a)
                .filter(e -> e % 2 == 0)
                .forEach(System.out::println);
    }
}
```
### Note:
**filter:** removes what we do not want, keep only what we want.

## 3. print the squares of all the even elements of the given list
```java
import java.util.Arrays;

class Main {
    public static void main(String[] args) {
        int[] a = {1,2,3,4,5,6,7,8,9};
        printNumbers(a);
    }

    private static void printNumbers(int[] a) {
        Arrays.stream(a)
                .filter(e -> e % 2 == 0)
                .map(e -> e * e)
                .forEach(System.out::println);
    }
}
```

## 4. Print only odd numbers from the list

## 8. Print the cubes of all odd numbers

## 9. Print the number of characters in each course name
```java
import java.util.*;
```
